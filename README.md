# TypeScript in One File: The Master Reference 🚀

This repository contains a comprehensive, 500+ line summary of **TypeScript**, covering everything from basic types to advanced Object-Oriented Programming (OOP).

### 📖 What's inside?
- **Basic & Advanced Types:** Tuples, Enums, Unknown, Never.
- **OOP Mastery:** Classes, Inheritance, Abstract Classes, Interfaces, and Encapsulation.
- **Generics:** Reusable components, Generic Interfaces & Functions.
- **Decorators:** Class & Method metadata.
- **Utility Types:** Omit, Pick, Partial, etc.

### 🎯 Why this repo?
I created this while building **Expertly AI** (a Full-stack project) to serve as a quick yet deep reference.
